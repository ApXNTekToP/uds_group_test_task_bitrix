## Тестовое задание
Тестовое задание для Битрикс24 разработчика

Написать компонент который выведет список элементов инфоблока.

Параметры компонента:

Выбор инфоблока - сделать так же, как это реализовано в стандартном компоненте news

Список полей для вывода через запятую

Список свойств для вывода через запятую

Кеширование
Список элементов инфоблока выводим через main.ui.grid

Должна работать пагинация

Должен работать выбор количества элементов на странице

Должно отображаться сколько всего элементов

Фильтр по таблице через main.ui.filter

При вводе в поисковую строку ищем по NAME и PREVIEW_TEXT и DETAIL_TEXT

Добавить фильтр по полю DATE_CREATE

Важно: Необходимо использовать новое api и D7 и orm. Компонент должен быть создан через class.php

## Тестовый компонент
Реализован компонент:
```
test:iblock.grid
```
Путь:
```
/local/components/test/iblock.grid
```

Компонент выводит список элементов инфоблока через: main.ui.grid, main.ui.filter

## Возможности компонента
Компонент поддерживает:
- выбор инфоблока (как в стандартном компоненте news)
- вывод полей элемента 
- вывод свойств элемента 
  - поиск по:
  - NAME 
  - PREVIEW_TEXT 
  - DETAIL_TEXT
- фильтр по DATE_CREATE 
- пагинацию 
- выбор количества элементов на странице 
- отображение общего количества элементов

## Тестовая страница
Компонент можно протестировать по адресу:
http://localhost:8080/test/iblock_grid.php

## Пример подключение компонента
```php
$APPLICATION->IncludeComponent(
    "test:iblock.grid",
    "",
    [
        "IBLOCK_TYPE" => "content",
        "IBLOCK_ID" => 1,
        "FIELD_CODES" => "ID,NAME,DATE_CREATE",
        "PROPERTY_CODES" => "",
        "CACHE_TIME" => 3600,
    ]
);
```
